## Story

**Como** dev mantenedor do youtube-biolink,
**eu quero** ESLint + Prettier + commitlint configurados com hooks locais,
**para que** o repositório mantenha qualidade de código e discipline didática de commits desde o início.

5. `pnpm lint` passa limpo no estado atual do projeto.

### Padrão de Commits

Conventional Commits: `feat(scope): descrição [Story X.Y]`. [Source: architecture.md § 16]

Exemplos válidos:
- `feat: add health route [Story 1.7]`
- `chore: setup husky hooks [Story 1.2]`
- `fix(lint): correct eslint config scope [Story 1.2]`
