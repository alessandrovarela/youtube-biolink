# Stories do youtube-biolink

Este diretório contém as stories de desenvolvimento do projeto, organizadas por epic.

## Convenção de nomenclatura

```
{epicNum}.{storyNum}.story.md
```

Exemplo: `1.3.story.md` = Epic 1, Story 3.

## Epics

| Epic | Título | Status |
|------|--------|--------|
| 1 | Fundação & Canary | Em andamento |
| 2 | Autenticação & Identidade | Planejado |
| 3 | Links & Página Pública | Planejado |
| 4 | Design System & Temas | Planejado |
| 5 | Analytics de Cliques | Planejado |
| 6 | Segurança em Camadas & Hardening | Planejado |

## Template

Stories seguem o template definido pelo AIOX em `.aiox-core/development/templates/story-tmpl.yaml`.

Cada story contém:
- Acceptance Criteria testáveis
- Tasks com checkboxes
- Dev Notes com referências canônicas
- Definition of Done
- File List (preenchido durante implementação)
- Change Log
